# Exercicio 06

Crie uma classe Empregado que terá como atributos:

    Identificação
    Nome
    Sobrenome
    Salário (mensal)

Crie métodos para:

    Saber o salário anual do empregado
    Saber o nome completo do empregado
    Modificar o salário, o parâmetro do método deve ser o percentual de aumento
